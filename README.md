# 건설 도면 탐색 인터페이스

## 실행 방법

```bash
npm install
npm run dev
```

> **참고**: `public/data/drawings/` 폴더에 도면 이미지 파일들을 배치해야 합니다.

## 기술 스택

- React 18
- TypeScript
- Vite 5
- Zustand (상태 관리)
- Tailwind CSS (스타일링)
- HTML5 Canvas API (도면 렌더링)

## 구현 기능

- [x] 공간 탐색 (전체 배치도 → 건물 선택)
- [x] 배치도 Hotspot 클릭 (건물 영역 클릭으로 이동)
- [x] 공종 선택 (건축, 구조, 설비, 소방 등)
- [x] Region 선택 (101동 구조 A/B 영역)
- [x] 리비전 타임라인 (이력 확인 및 전환)
- [x] 레이어 오버레이 (다른 공종 겹쳐보기, 투명도 조절)
- [x] Figma 스타일 레이어 드래그 (Shift + 드래그로 위치 조정)
- [x] 줌/팬 컨트롤 (마우스 휠, 드래그)
- [x] 리비전 비교 (나란히, 겹쳐보기, 슬라이더 모드)
- [x] Breadcrumb 네비게이션
- [x] 도면 정보 패널
